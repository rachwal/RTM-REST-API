What is RTM REST API?

RTM REST API is a free, open source Robotic Technology Middleware component that supports communication between OpenRTM platform and remote clients using REST API. You can use it to stream video from remote device to OpenRTM platform using HTTP protocol. Clients of this component can run anywhere, and they don't require you to use a particular programming language or framework.
